# SL Batch Tool

Набор инструментов для пакетного импорта и экспорта анимаций через Studio Library. Утилита автоматизирует рутинные процессы, опираясь на логику скрипта AnimAssistant.

## Возможности
* **Batch Export:** Автоматический экспорт списка клипов из AnimAssistant (включая создание плейбластов, `pose.json` и генерацию иконок) прямо в выбранную папку Studio Library.
* **Batch Import:** Автоматическая сборка секвенции анимаций из Studio Library в таймлайн Maya с сохранением временных интервалов и добавлением Idle-поз.

## Установка (Для пользователей)
1. Скачайте/скопируйте папку со скриптом к себе на компьютер (например, в `Documents/maya/scripts/`).
2. Откройте Maya.
3. Возьмите файл `install.py` из папки скрипта и **перетащите его прямо в рабочую область Maya (Viewport)**.
4. На вашей текущей полке появится новая кнопка (с подписью "SL_B").

## Использование
1. Нажмите на созданную кнопку на полке.
2. В интерфейсе Studio Library появятся две новые кнопки: **Btch_Import** и **Btch_Export**.
3. Выберите нужную папку внутри Studio Library и нажмите соответствующую кнопку для запуска пакетной операции.
